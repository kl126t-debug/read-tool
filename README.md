## 开发说明

这是一个纯，无需。所有

### 本地开发
1. 将所有文件放在Web服务器目录下
2. 通过HTTP(S)协议访问（不能使用file://协议）
3. 在浏览器开发者工具中

### 部署
可以部署到任何静态文件托管服务，如：
- GitHub Pages
- Netlify
- Vercel
- 或任何Web服务器
